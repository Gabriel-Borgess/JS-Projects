# JS-Projects
 Este repositório contém projetos em JavaScript, que abrangem uma variedade de conceitos e práticas de programação.

### 1 -[Chatbot HTML CSS JavaScript WITH GPT 3.5](./ChatBotJS)

ChatBotJS é um projeto de chatbot desenvolvido em JavaScript utilizando OpenAI GPT-3.5. O objetivo do projeto é criar um chatbot capaz de fornecer respostas inteligentes e interagir com os usuários em tempo real. Este chatbot também é responsivo, o que significa que você também pode usá-lo em seu telefone. 

Funcionalidades
Interface de chat intuitiva para interação com o chatbot.
Envio de mensagens do usuário para o chatbot.
Respostas geradas pelo chatbot com base na análise do contexto da conversa.
Integração com a API da OpenAI para obtenção das respostas do chatbot.
